# 개요

RC4 스트림 암호를 사용한 간단한 암/복호화 모듈입니다.
파일 명과 키를 인자로 넘겨, 암호화된 결과를 표준 출력합니다.
표준 출력을 리다이렉션을 통해 파일로 저장해 암/복호화된 파일을 생성할 수 있습니다.
RC4는 스트림 암호이기 때문에, 암호화 과정과 복호화 과정이 동일합니다. 따라서 복호화 시에는 암호화된 파일을 암호화에 사용한 키와 인자로 넘겨 복호화를 수행합니다.

# 사용법

다음 명령어로 프로그램을 컴파일합니다:
```
gcc RC4.c -o RC4
```

다음 명령어로 파일을 암/복호화합니다:
```
./RC4 <파일 명> <키>
```
